
## 依赖注入

- 仅适用被provide的后代才能使用

### Provide

```typescript
@Component({
  id: "useinfo",
  component: "form",
  properties: [
      Gender
  ]
})
class UseInfo extends Field {
  @Provide("optionList") // 当key 和提供的属性的key一致时，可以不用写
  options = [
    {
      value: "male",
      label: "Male"
    },
    {
      value: "female",
      label: "Female"
    },
    {
      value: "other",
      label: "Other"
    }
  ];
}
```

### Inject

```typescript
@Component({
  id: "gender",
  component: "select",
})
class Gender extends Field {
  @Inject("optionList") // 当key 和提供的属性的key一致时，可以不用写
  options = [];
}
```
